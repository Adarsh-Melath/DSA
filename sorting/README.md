## Sorting

-   Sorting is the process of arranging data in a specific order, usually ascending or descending.

## Common Sorting Algorithms

_Bubble Sort_ – Simple, compares adjacent elements

_Selection Sort_ – Selects the minimum element repeatedly

_Insertion Sort_ – Inserts elements into their correct position

_Merge Sort_ – Divide and conquer approach

_Quick Sort_ – Fast, uses pivot-based partitioning
